# Sigmoid_PC
Evaluation of a Power control algorithm for D2D underlaying cellular networks (see https://www.mdpi.com/2079-9292/8/10/1113)
This was my Final Year Project for my undergraduate degree in Electrical engineering from teh university of Malaya wherein I evaluate the power control algorithm or power update function derived in Al-Gumaei et al 2019 (https://www.mdpi.com/2079-9292/8/10/1113) and also evaluate some aspects of the cost function (sigmoidial).
